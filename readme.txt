The current folder contains the final Python code developed for the master's thesis of:
T.H.E. Gulikers
Delft University of Technology
Date: 24-11-2018

Two modules are presented: The data generator and the ANN training modules. To generate the UMAT, execute the 'createUMAT' function at the end of the ANN training script. a '.for' file is created that can be readily integrated in Abaqus as a constitutive model.
